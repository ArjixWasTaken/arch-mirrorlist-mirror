Arch Linux Mirror Status

This repository provides a regularly updated JSON file containing the status of Arch Linux mirrors.

Mirror Status JSON

Direct Link: mirrors.json

Raw URL: https://arjixwastaken.github.io/arch-mirrorlist-mirror/mirrors.json

Usage

You can fetch the mirror status data using:

curl https://arjixwastaken.github.io/arch-mirrorlist-mirror/mirrors.json

Data Source

The data is automatically fetched from https://archlinux.org/mirrors/status/json/ and updated daily via a cron job.
